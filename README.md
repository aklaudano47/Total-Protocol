# TOTAL Protocol 🧠📈  
**A Smart Contract–Deployed Macro Index Token**

`TOTAL` is a custom ERC20 token that simulates a synthetic macro index of the total crypto market cap. Designed for governance, integration, and long-term utility, this smart contract brings together multiple advanced features in one clean, gas-efficient implementation.

## 🚀 Contract Overview

- **Token Name:** TOTAL Macro Index  
- **Symbol:** TOTAL  
- **Deployed To (Sepolia):** [`0xcbAcE8eB126efbE38c31fCd0527a400b7b9BC0cc`](https://sepolia.etherscan.io/address/0xcbAcE8eB126efbE38c31fCd0527a400b7b9BC0cc)

## 🛠️ Features

- ✅ **ERC20**: Standard fungible token logic  
- ✅ **ERC20Votes**: Full on-chain governance support  
- ✅ **ERC20Permit**: Gasless approvals (EIP-2612)  
- ✅ **Ownable**: Role-restricted functions for the protocol owner  
- ✅ **Minted on deploy**: 1,000,000 TOTAL tokens minted to the deployer  

## 🧱 Tech Stack

- **Solidity v0.8.28**
- **Hardhat** (Ethereum dev environment)
- **OpenZeppelin Contracts v5.4.0**
- **Infura** (Ethereum node provider)
- **dotenv** for managing private keys securely
